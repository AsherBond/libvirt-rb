Overview

Represents an exceptional event within the Libvirt library. This contains an error readable attribute which, if available, is a Libvirt::Error object, which contains more details about the error which occurred.

Direct Known Subclasses

ConnectionFailed, DomainNotFound, InitializationError

def initialize(error=nil)
  # This works because error.message will be nil if there
  # wasn't an error.
  @error = error || Libvirt::Error.last_error

  message = error
  message = error.message if error.is_a?(Libvirt::Error)
  super(message)
end
